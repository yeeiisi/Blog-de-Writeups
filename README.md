# 🧠 Blog de Writeups - CTF & Cybersecurity

Este repositorio contiene una colección de **writeups de CTF (Capture The Flag)** y retos de ciberseguridad resueltos durante mi aprendizaje.

El objetivo principal es documentar el proceso de resolución, reforzar conocimientos técnicos y compartir soluciones detalladas en áreas clave de la seguridad informática.

---

## 🎯 Propósito

Este blog existe para:

- 📌 Documentar soluciones a retos de CTF
- 🧠 Explicar el razonamiento paso a paso
- 📚 Servir como material de estudio personal y público
- 🚀 Construir experiencia práctica en ciberseguridad
- 🔍 Mejorar habilidades en hacking ético y análisis de vulnerabilidades

---

## 🧩 Áreas de ciberseguridad cubiertas

- 🔐 Cryptography
- 🧠 Reverse Engineering
- 💣 Binary Exploitation (Pwn)
- 🌐 Web Security
- 🧪 Forensics
- 🛠️ Misc / General challenges

---

## 📂 Estructura del repositorio

Cada carpeta corresponde a un CTF o evento específico:
- CTF_NAME/
  - challenge-name/
    - writeup.md
    - files/
      - exploit.py (si tiene)


Cada writeup incluye:

- Descripción del reto
- Análisis inicial
- Metodología de resolución
- Exploit o solución final
- Aprendizajes obtenidos

---

## 📖 Writeups destacados

Algunos tipos de retos incluidos:

- 🔹 Criptografía (RSA, XOR, hashes, encoding)
- 🔹 Explotación de binarios (buffer overflow, heap)
- 🔹 Reverse engineering con Ghidra/IDA
- 🔹 Vulnerabilidades web (SQLi, XSS, LFI, etc.)
- 🔹 Análisis forense de archivos y tráfico

---

## 🛠️ Herramientas utilizadas

Durante los retos se utilizan herramientas como:

- Python 🐍
- pwntools
- GDB / pwndbg
- Ghidra / IDA Free
- Burp Suite
- CyberChef
- Wireshark

---

## 🧠 Cómo navegar este repositorio

1. Selecciona un CTF o carpeta de evento
2. Entra en el reto específico
3. Lee el writeup paso a paso
4. Revisa scripts o exploits incluidos
5. Analiza el razonamiento aplicado
